## Notebook

This folder contains a Jupyter notebook used to generate the synthetic dataset.

### File

* `data_generation.ipynb`
  Builds a synthetic dataset with realistic patterns such as:

  * seasonality
  * regional variation
  * product-level effects
  * random noise

### Purpose

The notebook demonstrates how the dataset was constructed and ensures
reproducibility of the analysis.

### Notes

* The generated dataset is saved as `synthetic_dataset.csv`
  in the `/data` folder
* The notebook focuses on data simulation, not analysis
