## Data

This folder contains a synthetic dataset used for demonstration purposes.

### File

* `synthetic_dataset.csv`
  Simulated dataset representing tool hire utilisation, forecasts,
  and related attributes.

### Notes

* The dataset was artificially generated to mimic the structure of a
  real production dataset.
* Only a subset of fields required for the analysis is included.
* Data covers a limited time period (~3–4 months) for simplicity.

### Usage

This dataset can be uploaded to BigQuery and used as a source table
for SQL scripts located in the `/sql` folder.
